ROOT takes the BS out of ROBOTS

Other coding robots are one experience for one age group and toys like these sit on a shelf after a few weeks. Root is the only robot in the world that can be used effectively to teach coding from Kindergarten to College. Learning opportunities with Root will last years, not days.

Graphical Programing

A graphical interface that beginners of any age (even non-readers) can understand. Colored blocks reinforce its basic "if this, then that" framework. It teaches events, sequences, loops, states, functions, peiorities, timing, program stepping, and debugging.

Computional Fluency

Introduces advanced flow control statements like when, repeat, and if-then-else, in order to foster computational fluency. It teaches variables, sensor values, units, arithmetic operations, recursion, and parallelism.

Full text programming

A fully text-based interface that takes programmers to professionally used languages including Python, JavaScript, and Swift. It teaches users to text equivalent of code from the previous levels, and lets them build powerful programs to accomplish new goals.

Program at different levels

Switch between three levels of proficiency - graphical blocks, computational constructs, and text programming - aiding the transitions as mastery increases.

Code interactivity

No compiling necessar; programs can be modified even while they are running. Users can flexibly pause, step, or add instructions at any time.

See what the robot sees

Giving users constant access to what the robot knows promotes agent-based thinking and facilitates the debugging of robot programs.

"If this, then that"

The first level of Root Square provides a natural, intuitive way to let the robot take actions in response to events and prioritize thise responses.

Wireless inputs

Wireless communication lets the robot respond as though a phone or tablet's sensors were a part of itself.

Share programs

Users can save and share their programs with each other to create a network with an unlimited number of new activities.
